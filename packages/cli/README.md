# @uclaw/cli [EXPERIMENTAL]

> [!WARNING]
> This package is currently under active development, considered **experimental**, and is **not** ready for general production use.

## Overview

`@uclaw/cli` is the command-line interface for the UClaw platform. It is designed to assist developers with managing their AI agents, running interactive sessions in the terminal, and deploying agentic applications.

All CLI features (including `uclaw run`, `uclaw app`, etc.) are currently for INTERNAL use. Do not use in production.

## Status

- **Stability**: Experimental (Pre-Alpha)
- **Status**: Internal testing only.
- **Usage**: Not recommended for external installations.

If you are looking to build agentic applications, please use the official, production-ready SDK instead:

- [@uclaw/sdk](../sdk)
